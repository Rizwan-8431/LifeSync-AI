# 🚀 LifeSync AI

LifeSync AI is a Personal Growth, Habit Tracking, and Productivity Companion built with Flask and Gemini AI.

The platform helps users improve their daily productivity, manage tasks, track personal goals, maintain journals, analyze progress, and receive AI-powered guidance for self-improvement.

---

## ✨ Features

### 👤 User Onboarding

* User profile creation
* Age, occupation, and goal setup
* Personalized dashboard experience

### 📋 Task Management

* Add tasks
* Complete tasks
* Delete tasks
* Progress tracking
* Completion analytics

### 🤖 AI Coach

* Gemini-powered AI assistant
* Personalized productivity advice
* Goal-based recommendations
* Learning and self-growth guidance
* Motivation and habit-building support

### 📖 Reflection Journal

* Daily reflections
* Learning notes
* Personal growth tracking

### 📊 Analytics Dashboard

* Task completion tracking
* Productivity insights
* Weekly progress reports (Upcoming)

### ⚙️ Settings

* User profile management
* Personalization options

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Rizwan-8431/LifeSync-AI.git
```

Move into the project directory:

```bash
cd LifeSync-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

---

## 🔑 Environment Variables

Create an environment variable:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Never store API keys directly in source code.
